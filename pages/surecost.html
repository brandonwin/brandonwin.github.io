The process behind the redesign

I designed a prototype of a dashboard interface for SureCost's web app as well as revamped navigational menus. The driving concept behind my design was to preserve the current ways users interact with the design (one notable interaction was keyboard shortcuts for Purchasing, Formulary, etc.) while adding new, useful affordances to streamline user workflows (eg. search bar, editable layouts, help wizard).
