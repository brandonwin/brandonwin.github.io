CollabApp

Collaboration tools emphasizing communication, resource sharing, and accountability.

Click this to view high-fidelity designs.

CollabApp is a collaboration tool designed for short-term or long-term projects, and it can be used in an academic or professional environment. The interface I designed is heavily borrowed from that of Discord, but I also have considered CollabApp to be a mod for Discord itself.

I took inspiration from online collaboration tools such as Slack and Github, with the dashboard interface focusing on users being able to mention each other and add comments for changes to the project that they made.

The dashboard is designed to be modular, with the user being able to add, move or delete panels. The header panel is designed to never change regardless of the current page that they're on, so the user always has access to the calendar, push notifications and search bar.

The interface was designed completely in Adobe XD, with some work done in Illustrator or Photoshop to design icons or touch up images. I also created a user journey map before the design prototype to know what was needed for the user.
