CollabApp

Problem: Design an application for students to collaborate on team projects with.

Solution: For my UX Design course, I designed CollabApp. CollabApp is a desktop and tablet app designed for college students to outline, manage and finalize short-term and long-term group projects. I took visual inspiration from Discord and Slack, and functional inspiration more broadly from dashboard designs in various apps. I believed that Discord and Slack's organizational structure would complement student group projects, and I used their model as an outline for the dashboard design for the application. I organized each major section of a group project (calendar, checklist, common resources, etc.) into separate groups to make it easier to navigate through the separate functions for students.

CollabApp is a collaboration tool designed for short-term or long-term projects, and it can be used in an academic or professional environment. The interface I designed is heavily borrowed from that of Discord, but I also have considered CollabApp to be a mod for Discord itself.

I took inspiration from online collaboration tools such as Slack and Github, with the dashboard interface focusing on users being able to mention each other and add comments for changes to the project that they made.

The dashboard is designed to be modular, with the user being able to add, move or delete panels. The header panel is designed to never change regardless of the current page that they're on, so the user always has access to the calendar, push notifications and search bar.

The interface was designed in Adobe XD, with some work done in Illustrator or Photoshop to design icons or touch up images. I also created a user journey map before the design prototype to know what was needed for the user.
