PROJECT: Alone

A curated Web x Music experience.

PROJECT: Alone is an online website that contains a curated list of music relating to the feelings of loneliness, heartbreak, death and more that many people may have related to during the COVID-19 pandemic.

Product

Website

Technologies

HTML, CSS, JavaScript, Bootstrap

Key Services

Web Design, UI Design, Responsive Design, Scripting, Front-End Development, Wireframing

The Problem

Being heavily interested in music analysis, I wanted to create some form of online project that would be able to show a variety of emotions and relate them to curated songs.

The Solution

I decided to create an online website that had several songs explaining their meaning and relation to an overall arcing emotion. This website would be longform, allowing the user to scroll through the curated list.

Iteration

Web Design

I designed the site with HTML, CSS, and JavaScript to build the layout and song modules. I primarily utilized Bootstrap and CSS3 AnimateIt to build the website layout and animations for the song modules, configuring their visuals to create a user-accessible and enjoyable listening experience.

Copywriting

Copywriting is a large portion of the project. I wrote the copy for each section/emotion, and I found myself constantly editing and perfecting the copy to make it match the overall theme of the project.
