PROJECT: Alone is an interactive music site that showcases music I listened to during the pandemic. I built the website with HTML/CSS and Bootstrap in a week for a web design course.

My role

Design website layout
Build Bootstrap and music player features
Copywriting

Results

Website built
Music player implemented

Problem

I wanted to curate music that would relate to the ongoing pandemic and explain those songs.

Solution

Create an online website that showcased certain songs and their meanings in relation to the overall theme.

Booting the Bootstrap

I built the site with HTML, CSS, and JavaScript, and this was my first venture into using Bootstrap. I utilized Bootstrap and CSS3 AnimateIt to build the song modules and animations, configuring their visuals to create an accessible and enjoyable listening experience.

Copywriting

Copywriting was a large portion of the project, and I chose the tone and style of the copy. I wanted to make sure that each song's description still fit the overall theme of the page, and I iterated on the copy many times to make it click.
