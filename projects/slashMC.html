slashMC

Problem: Redesign the clunky, outdated graphical user interface for operating a Minecraft server.

Solution: I designed slashMC, a desktop and mobile app that offers a sleek, informative interface that allows Minecraft server operators to have all of the power of text-based server operators with the sleekness of a modern interface. slashMC has all of the functionality of more technical server interfaces, but I designed it to be more accessible and user-friendly to the average users. I also designed slashMC GO, a mobile companion app, which allows server operators to view and manage their server on the go, something that I believe differentiates my prototype designs from other competitors.

slashMC is my take on redesigning the Minecraft server GUI, which I found to be outdated and unattractive, while providing very little benefits over other low resource-intensive options. From lessons learned from running my own Minecraft servers to play with friends, I designed slashMC to be not only appealing to new server operators, but also useful and powerful to returning operators.

slashMC features an interface that borrows elements from the vanilla server GUI, but also has additional interactions and contextual actions that are more in-line with modern applications. For example, the server console panel's command lines are able to be right-clicked on, and depending on what you right click, you are given different options (e.g. mentioning a user, re-directing to a forum page with troubleshooting). Other features include a search bar, in-app help database, and improved settings management.

One core feature I wanted slashMC was the app's ability to change and apply the server properties file based on which world file is currently loaded in. If you're familiar with Minecraft server operation, you would know that the server properties file applies to your server globally, ignoring whatever world file you're in. I found this to be outdated, and slashMC has an interface that directly tackles this issue.

Another feature that I found current Minecraft server operation lacking is the addition of a mobile companion. Therefore, I designed a mobile companion that has the ability to send commands to your computer wirelessly.
